This repository contains my Equity Research Reports covering fundamental analysis of listed companies. The reports include detailed study of financial statements, ratio analysis, industry comparison, valuation techniques (DCF, relative valuation), and investment recommendations.

The objective of this repository is to develop strong analytical skills in finance and build a professional portfolio for roles in investment banking, equity research, and financial analysis.

Tools and concepts used include financial modeling, Excel analysis, macroeconomic evaluation, and business strategy assessment.
