## SAIL Equity Research Report

This report presents a detailed fundamental analysis of Steel Authority of India Limited (SAIL) using an FCFE-based Discounted Cash Flow (DCF) valuation model.

### Key Highlights:
- Intrinsic Value: ₹190.55
- Market Price: ₹154.94
- Upside Potential: ~18–25%
- Recommendation: BUY

### Methodology:
- Beta estimation using regression
- Cost of Equity via CAPM
- WACC computation
- FCFE-based valuation
- Sensitivity & scenario analysis
